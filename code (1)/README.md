# IT Corporate & Portfolio Website

A modern, responsive corporate introduction and portfolio website built with Next.js, Tailwind CSS, and Framer Motion.

## Features

- **Interactive Hero Section**: Dynamic particle background and animations.
- **Project Showcase**: Filterable portfolio with parallax effects.
- **Client Request Showcase**: Unique comparison of client requests vs. final results.
- **Responsive Design**: Optimized for all devices.

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion
- Supabase (Backend/Database)
- Radix UI

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Set up `.env.local` with your Supabase credentials.
4. Run dev server: `npm run dev`
